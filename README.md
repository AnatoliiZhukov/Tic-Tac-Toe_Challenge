Some UI elements need TextMeshPro which is not included in the repository.

The challenge was a part of the Game Engine 2 course at JAMK UAS in 2023.

When a symbol is placed on the grid, the game checks if the player has won by comparing all placed symbols to an array of all possible win conditions. If all cells have a symbol in them and no player has won, the game ends in a tie.
